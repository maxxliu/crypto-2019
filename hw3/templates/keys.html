Generate Keypair

Press the button to generate a new public/private keypair and a related GoodCoin address. Note that the address is simply a base58 encoding of your public key, so only the address and the private key (also base58 encoded for your convenience) are included here. When creating new transactions using this keypair, you will only need to provide the address and private key in order to properly sign your transaction. This is because you are including the public key implicitly when you provide the address, since the address is just a more user-friendly encoding of the same information. We choose to use the term address here rather than public key in order to be consistent with the terminology used in Bitcoin and other mainstream cryptocurrencies.
